# Simple Turing Machines, Statistic Complexity vs Algorithmic Complexity

## INSTALL
Get TMCompression and make the project, using:
```bash
git https://github.com/jorgeMFS/TMCompression.git
cd TMCompression
make
make ioStNormalize
```
## RUN

```bash
chmod +x *.sh
bash ProcessResults.sh 1 0 0 0
```
